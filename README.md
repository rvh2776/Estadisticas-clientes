##Gráficos zona clientes.


Plugin que crea una pagina en la zona del cliente, muestra estadísticas del consumo en tiempo real.

Plugin that creates a page in the client zone, shows consumption statistics in real time.

------------
####**Descargar e instalar usando la interfaz de plugins de UCRM.**

------------


## **Configuración.**

**Período de ejecución** – dejar como está, UCRM lo maneja automáticamente.

| Nombre de campo | Configuración |
| :------------: | :------------: |
| Nombre de dominio  | Ingrese su nombre de dominio ej: https://ucrm-demo.ubnt.com|
| API token UNMS  | Ingrese el 'API token' creado en: nms/settings/users.  |
|  Nombre de empresa | Nombre que se muestra abajo a la derecha en el grafico ej: © MyCompany ISP.  |
| Cambiar título de la página  | Traducir/cambiar el título de la página ej: Estadísticas, Statistics. |
| Cambiar título del gráfico  | Traducir/cambiar el título del grafico ej: Cliente: Juana de Arco, Client: Juana de Arco.  |
| Cambiar descripción del pie  | Traducir/cambiar descripción del grafico ej: Últimos 60 minutos, actualizados cada 3 segundos. |
| Cambiar descripción vertical  | Traducir/cambiar descripción vertical del grafico ej: Velocidad: kbit/segundos. |
| Cambiar descripción de la serie bajada  | Traducir/cambiar la descripción de la serie bajada ej: Bajada, Download.  |
| Cambiar descripción de la serie subida  | Traducir/cambiar la descripción de la serie subida ej: Subida, Upload.  |
|Menú exportar gráfico| Activa o desactiva el menú de exportar los gráficos.  |
